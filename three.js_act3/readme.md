# Interactive Physics Scene (Three.js + Cannon.js + GLTFLoader + Raycaster)

## Runs with NO cannon-es install
Cannon is loaded via CDN in index.html.

## Model path used
`./static/models/Duck/glTF-Binary/Duck.glb`

## Run
Use any local server (recommended):
- VS Code Live Server (right click index.html -> Open with Live Server)
OR
- Vite if required by class:
  npm install
  npm run dev

## Controls
- Hover: highlight
- Click: push object
- Strong impact: WebAudio thud
